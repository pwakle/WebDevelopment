The border-radius property defines the radius of the element's corners. Border radius helps create rounded borders for elements like buttons or images.
The border color property sets the colour of the border.
Border Width Specifies the width of the border. Sets the width of the border in pixels(px), or there are values like medium, thin, and thick to set the border width.
