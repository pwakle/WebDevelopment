1 Box Model Width, Height, and Margin

Everything in CSS has a box around it, and understanding these boxes is key to being able to create more complex layouts with CSS

The CSS box model is essentially a box that wraps around every HTML element. It consists of: borders, padding, margins, and the actual content

The innermost component of the box model is the actual content of the element. It can be text, image, video, etc. The content area is defined by the width and height properties.
What is padding?
Padding is the the space between the actual content and the border of the element
The border surrounds the content and padding and gives the visual border of the element.
The margin is the space outside the element that separates it from other elements in the layout.
